9. Mutt

Mutt is a powerfull mail reader, Mutt Manual, is the best place to find information. I recomend Jump Start: Mutt, name says it all.

        $ prt-get depinst mutt

9.1. Configuration

Read Justin R. Miller Mutt Gnupg Howto "Everything You Need To Know To Start Using GnuPG with Mutt".

Mutt will recognize two locations for its configuration file; ~/.muttrc and ~/.mutt/muttrc. Skeleton configuration will use the second one.

        $ mkdir /etc/skel/.mutt

Get gpg.rc example from distribution file;

        $ tar --extract --file=/var/ports/distfiles/mutt-1.5.24.tar.gz \
        > mutt-1.5.24/contrib/gpg.rc
        $ sudo mv mutt-1.5.24/contrib/gpg.rc /etc/skel/.mutt/
        $ sudo touch /etc/skel/.mutt/mail_alias
        
        $ sudo vim /etc/skel/.mutt/muttrc

Change pgp_sign_as to 0xyour_pub_key;

        set config_charset="utf-8"
        set charset="utf-8"
        set send_charset="utf-8"
        set editor="vim"
        set sort=reverse-date-received

        source ~/.mutt/gpg.rc

        set pgp_autosign=yes
        set pgp_sign_as=0x29CA16D3
        set pgp_replyencrypt=yes
        set pgp_timeout=1800

        # automatically sign all outgoing messages
        set crypt_autosign

        # sign only replies to signed messages
        set crypt_replysign

        # automatically encrypt outgoing messages
        set crypt_autoencrypt=yes

        # encrypt only replies to signed messages
        set crypt_replyencrypt=yes

        # encrypt and sign replies to encrypted messages
        set crypt_replysignencrypted=yes

        # automatically verify the sign of a message when opened
        set crypt_verify_sig=yes

When listing messages the status flag mean;

    s
    if the message is signed and not yet verified
    S
    if the message is signed and the signature is successfully verified
    P
    if the message is PGP encrypted

9.1.1 Alias

Alias makes easy to manage email addresses. Add this to your muttrc;

        source "~/.mutt/mail_alias"
        set alias_file=~/.mutt/mail_alias

While on index or page press "a" to add address to alias file.

9.2. Local Email

        $ mutt
        /home/admin/Mail does not exist. Create it? ([yes]/no): yes

After this I get the following error;

        /var/spool/mail/admin: No such file or directory (errno = 2)

Press q or x and lets send email to root, this way we test if alias is working and exim create mail dir for us;

        $ mutt -s "test" root@localhost < /dev/null

9.2.1. Tagging Email

Just press shift-T and then read Patterns . After matching desired messages press ; then the order, for example, d will mark all taged for deletion.

9.3. POP and SMTP

If you want to use built in SMTP create a copy off official opt/mutt to enable SMTP and sasl. Alternative is to use external programs like msmtp.

        $ prt-get cyrus-sasl 
        $ cp -R /usr/ports/opt/mutt .
        $ cd mutt
        $ vim Pkgfile

Uncomment the following line;

        --enable-smtp --with-sasl

Build your personalized mutt;

        $ pkgmk -um
        $ sudo -H -u pkgmk fakeroot pkgmk -d 
        $ pkgadd -u /var/ports/packages/mutt#1.5.24-1.pkg.tar.gz
        

        
Configuration example of remote pop and smtp account; 


        
        set mbox_type=maildir
        set folder="~/MailExt/"
        set spoolfile=+/

        set realname='User'
        set from=user@remote.com
        set use_from=yes

        set ssl_starttls = yes

        set pop_user="user@remote.com"
        set pop_pass="password"
        set pop_delete=yes

        set pop_host="pops://$pop_user:$pop_pass@pop.remote.com:995"

        set smtp_url="smtps://$pop_user:$pop_pass@smtp.remote.com/"
        


        
9.4. Multiple Accounts


        
Multiple email accounts are gracefully handle by folder hooks,
        this hooks are defined in muttrc. When we change folder the
        hook is called, setting up the email account sepecific settings.


        
Lets configure two accounts, one for system related emails and 
        another for external email account;


        
        set config_charset="utf-8"
        # set locale="de_CH"
        set charset="utf-8"
        set send_charset="utf-8"
        set editor="vim"
        set auto_tag=yes
        set sort=threads

        set sort_browser=reverse-date
        set sort_aux=reverse-last-date-received
        set duplicate_threads=yes

        source ~/.mutt/gpg.rc

        set pgp_autosign=yes
        set pgp_sign_as=0x29CA16D3
        set pgp_replyencrypt=yes
        set pgp_timeout=1800

        # automatically sign all outgoing messages
        set crypt_autosign

        # sign only replies to signed messages
        set crypt_replysign

        # automatically encrypt outgoing messages
        #set crypt_autoencrypt=yes

        # encrypt only replies to signed messages
        set crypt_replyencrypt=yes

        # encrypt and sign replies to encrypted messages
        set crypt_replysignencrypted=yes

        # automatically verify the sign of a message when opened
        set crypt_verify_sig=yes

        source "~/.mutt/mail_alias"
        set alias_file=~/.mutt/mail_alias

        # Header
        set header_cache =~/.mutt/cache/headers  
        set message_cachedir =~/.mutt/cache/bodies  
        set certificate_file =~/.mutt/certificates

        ignore *
        unignore from: date subject to cc
        unignore x-mailing-list: posted-to:
        unignore x-mailer:

        folder-hook . 'unset sendmail; \
                unset pop_host; \
                unset smtp_url; \
                unset pop_user; \
                unset from; \
                unset pop_pass'

        # Local system account 
        folder-hook mail_system/* source ~/.mutt/system

        # Remote account
        folder-hook mail_external/* source ~/.mutt/external

        # Default account 
        source ~/.mutt/system

        ## Shortcuts
        macro index,pager  'source ~/.mutt/system!'
        macro index,pager  'source ~/.mutt/external!'

        set timeout=60	#Check for mail every minute
        set mail_check=5
        


        
Content of ~/.mutt/system;


        
        color status green default
        set mbox_type=maildir
        set folder="~/mail_system"
        set postponed="+postponed"
        set record="+sent"

        set spoolfile=/var/spool/mail/user
        set sendmail=/usr/sbin/exim

        set realname='Local User'
        set from=user@host.localhost
        set use_from=yes
        


        
Edit ~/.mutt/external and set with values of your server.


        
        color status blue default
        set mbox_type=maildir
        set folder="~/mail_external"
        set postponed="+postponed"
        set record="+sent"

        set spoolfile="~/mail_external"

        set realname='Realhuman'
        set from=realhuman@external.pt
        set use_from=yes

        set ssl_starttls=yes

        set pop_user="realhuman@external.pt"
        set pop_pass="password"
        set pop_delete=yes

        set pop_host="pops://$pop_user:$pop_pass@pop.external.org:995"
        set smtp_url="smtps://$pop_user:$pop_pass@smtp.external.org/"
