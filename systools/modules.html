Kernel modules

Edit /etc/rc.modules so it contains the modules needed by host hardware, this example load ath9k and r8169;

        #!/bin/bash
        #
        # /etc/rc.modules: module initialization script
        #

        /sbin/modprobe ath9k
        /sbin/modprobe r8169

        /sbin/depmod -a

        # End of file
