4. Vim

Read A good vimrc for more information. Global configuration file

/usr/share/vim/vimrc
System wide Vim initializations.
~/.vimrc
Your personal Vim initializations.

Create a vimrc skeleton to be copied by useradd to each user home directory;

        $ sudo vim /etc/skel/.vimrc
        
        if &term =~ '256color'
            " disable Background Color Erase (BCE) so that color schemes
            " render properly when inside 256-color tmux and GNU screen.
            " see also http://blog.greduan.com/2014/09/tbtbivtp.html
            set t_ut=
        endif

        if &term =~ '256color'
            set t_ut=
            set t_Co=256
        endif

        syntax on
        colorscheme desert

        set nowrap                      " don't wrap lines
        set tabstop=8 softtabstop=4 shiftwidth=4      " a tab is two spaces (or set this to 4)
        set expandtab                   " use spaces, not tabs (optional)
        set backspace=indent,eol,start  " backspace through everything in insert mode

        set autoindent  " Turns it on

        map >F2< :tabnew 
        map >F3< :tabprevious<CR>
        map >F4< :tabnext<CR>
        " Show Line Numbers
        set nu

        set complete=.,b,u,]
        set wildmode=longest,list:longest
        set completeopt=menu,preview

        set spell spelllang=en_us

Color Scheme 256

        $ wget -O desert256.vim http://www.vim.org/scripts/download_script.php?src_id=4055
        $ mkdir -p .vim/colors
        $ mv desert256.vim .vim/colors

Edit .vimrc;

        syntax on
        colorscheme desert256
