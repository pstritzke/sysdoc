5. Vim

<leader> with default configuration is key \, so when you see <leader>-W means pressing \W

5.1. Vim RC

Read A good vimrc for more information. Global configuration file

/usr/share/vim/vimrc
System wide Vim initializations.
~/.vimrc
Your personal Vim initializations.

Copy vimrc skeleton example, so that each user have a base to start personalizing it;

        $ sudo cp ~/sysdoc/conf/etc/skel/.vimrc /etc/skel/
        $ sudo mkdir /etc/skel/.vimswap
        $ sudo mkdir /etc/skel/.vimviews
        $ sudo mkdir /etc/skel/.vimundodir
        $ sudo mkdir /etc/skel/.vimbackup
        $ sudo mkdir /etc/skel/.vim
        $ wget -O desert256.vim http://www.vim.org/scripts/download_script.php?src_id=4055
        $ mv desert256.vim /usr/share/vim/colors/
