Getmail

        $ prt-get depinst getmail

Vim .getmail/getmailrc

        [retriever]
        type = SimplePOP3SSLRetriever
        server = pop.external.com
        username = user@external.com
        port = 995
        password = password

        [destination]
        type = Maildir
        path = ~/MailExt/

        [options]
        read_all = False
        message_log = ~/.getmail/log_external

read_all prevents all mail being downloaded every time getmail is ran.

Now, run getmail. If it works fine, you can create a cronjob for getmail to run every n hours/minutes. Type crontab -e to edit cronjobs, and enter the following:

        */1 * * * * /usr/bin/getmail --quiet --rcfile getmailrc

That will quietly run getmail every minute
